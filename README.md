Asteroids Deluxe

Originally developed for personal training by following the Boot.dev DevOps course.  

Requirements

    Python: 3.12 or higher.
    Pygame CE: The community edition of Pygame.

Installation & Running
Windows, macOS, and Linux

This project uses uv and runs with a single command without manually setting up a virtual environment.

    Clone the repository:

    git clone https://github.com/jtpoplin/asteroids.git
    cd asteroids

    Run the game:

    uv run main.py

Controls

    W/S: Move Forward / Backward
    A/D: Rotate Ship
    Space: Single-shot main weapon
    L: Piercing Laser
    K: Pulse Bomb
    J: Hellfire (Global Strike)
    ;: Triple-shot

Game generates locally besttime.txt and highscore.txt files for stat tracking.
